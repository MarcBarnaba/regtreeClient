Manuale utente

Avvio attraverso Jar eseguibile


E' possibile avviare sia il Server che il Client utilizzando i relativi file.jar e utilizzando gli argomenti previsti:

java -jar MapBCServer.jar [PortNumber]
java -jar MapBCClient.jar [NomeServer, PortNumber]



Server


Avviata l'applicazione, il server conferma la sua attività visualizzando una stringa a video, e in generale fornisce sempre un feedback su quello che sta avvenendo o è avvenuto.

Esempi:


Started: ServerSocket[addr=0.0.0.0/0.0.0.0,localport=8080]

Connessione accettata: Socket[addr=/127.0.0.1,port=65358,localport=8080]

Starting data acquisition phase!

closing...


Client


Avviato, il client permette all'utente di scegliere se costruire l'albero di regressione a partire da una tabella del database o da una già analizzata e presente in archivio.

Learn Regression Tree from database [1]
Load Regression Tree from archive [2]

File name:

Una volta inserito il nome, il sistema effettuerà le operazioni necessarie a iniziare la fase di previsione. In questa fase l'utente deve digitare il numero corrispondente al percorso che vuole seguire fino al nodo foglia dell'albero, il classvalue cercato.

Starting data acquisition phase!
Starting learning phase!
Starting prediction phase!
0:X=A
1:X=B

Al termine, il sistema visualizza il classvalue per la regola costruita e permette di ripetere la fase di previsione oppure terminare l'esecuzione.
